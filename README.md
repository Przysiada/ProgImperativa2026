# **Programação Imperativa**
#### Felipe Przysiada

---
# Descrição
A disciplina, de natureza teórico-prática, é ofertada no terceiro período do curso de Engenharia de Computação. O estudante aplica os conceitos de programação imperativa utilizando a linguagem C padrão com novos recursos em relação à disciplina anterior de RCE (Raciocínio Computacional para Engenharia, onde há uma introdução ao C) e a linguagem Python (apresentada como alternativa adicional de linguagem imperativa) na resolução de problemas computacionais estruturados. Ao término da disciplina, o estudante é capaz de utilizar tanto a linguagem C como a linguagem Python, conforme adequação, com desenvoltura de forma a criar sistemas considerando as funcionalidades  e recursos específicos das respectivas linguagens. Para frequentar a disciplina é necessário o conhecimento de lógica de programação e construção de algoritmos, desenvolvidos na disciplina de RCE.

---
# Objetivos
Construir soluções que resolvam problemas de engenharia utilizando algoritmos implementados na linguagem imperativa C; </br>
Construir soluções que resolvam problemas de engenharia computacional utilizando algoritmos implementados no paradigma imperativo em Pythom; </br>

---
# Temas
Compreender e aplicar o princípio e utilização da recursividade e modularização usando uma linguagem de programação; </br>
Compreender e aplicar ponteiros usando uma linguagem de programação; </br>
Compreender e aplicar os conceitos de alocação de memória, usando uma linguagem de programação; </br>
Compreender e aplicar o conceito arquivos para a solução de problemas usando uma linguagem de programação; </br>
Compreender e aplicar o conceito de busca para a solução de problemas usando uma linguagem de programação; </br>
Compreender e aplicar ordenação usando uma linguagem de programação; </br>
